写在最前面

发表于 2020-10-16 更新于 2020-11-01 分类于 生活 阅读次数： Valine：

这是最好的时代，也是最坏的时代。这是个知识爆炸的时代，知识的生产速度越来越快；同时，伴随着移动互联网的浪潮，人们获取知识更加方便，途径越来越多，获取知识的门槛极大的降低。举个我自己的例子，记得7年前的我，第一次离开家乡小城，在大学的第一次选拔面试中，老师问我认为学习电气自动化有什么意义，我绞尽脑汁也只能想到学会电气可以给自家的老房子修配电路。也是那一年，知乎刚刚取消邀请码制度，在一个个迷茫的夜晚，我可以通过一部手机，了解到大千世界，迅速的拓宽了我对世界的认识。不满足课堂上学到的枯燥的基础知识，我跟随着知乎上大牛的指引，自学了 Java，Java web，安卓编程，Linux，图像识别，ROS系统 ….

然而，躁动不安的青春，遇上滚滚的知识洪流，结果是样样都了解一些，但都不精通。4年前的我，因为自己碎片化的知识而感到万分苦恼，抱怨这是个最坏的时代。当时我觉得，计算机行业的自学门槛太低，只要一台电脑就可以学习加实践，自己应该选择一个自学门槛相对高的，并且更需要沉淀的专业去锤炼自己。机缘巧合，我最终选择了电力电子方向去读博士。然而，在这知识爆炸的时代，学术圈对“新知” 的狂热，更加无以复加，周围圈子里的博士，动不动就是好几篇SCI。虽然我的导师非常nice，几乎从来不push 我们，但在周遭氛围的影响下，对于没有研究生经历的我，出于自尊，也是为了毕业、生存，不可避免的，需要在这样压力的裹挟下，拼命向前奔跑着。过去的三年，为了成果的数量，经常是遇到真正的难点就绕开，很多东西都学的不扎实，分析的不彻底。时间飞逝，转眼到了博士学制的最后一年。今年以来，可能也是疫情的关系，在更多的 social distance 之下，我有了更多的反思自己的机会。我给自己制定了十年的规划，逐渐学会让自己在嘈杂纷繁的环境下多与自己对话，坚持去做自己。我发现，这是最坏的时代，坏到比过去任何一个时代都难以静下心来；然而，当你克服心魔、走出浮尘，能够静心追寻自我，这又会是一个最好的时代，因为你会发现，没有一个时代的普通人，会像今天这样有如此多的机会去提升自我，会如此宽容的让人们自由的选择自己的道路。

回想起来，真的好久没有写过如此长段的文字了，也因此写了非常久的时间，不过挺值得的，应该多给自己一些这样和自己对话的机会。最后，想说一下做这个博客的动机，主要就是两点，一是作为自己的总结和归纳，近几个月其实也写了一些总结的文档，基本都是用markdown 写的，使用 vscode + Markdown preview enhanced (vscode 的一个插件) 写起来还是挺顺手的。但是导出来的 pdf 还是太丑了点，看了一下用 hexo + GitHub pages 搭建还是挺方便的，自己看起来也漂亮很多。第二点也是希望多做一些分享，自己一直觉得受益于互联网时代，但大多时候还是作为一个知识、信息的汲取者，自己贡献的东西很少，写点东西，也是希望能够给需要的同学、朋友一些帮助。现在计划来说，这个 blog 应该主要是记录一些电力电子方向的基础知识，以及自己读文章的一些总结；此外更多的还会记录自己自学本专业以外知识的一些过程和总结。最后，希望以后也多写一些读书笔记和想法之类的东西，就像照片能够记录生命中某一刻的影像，而只有文字，是灵魂思索的片刻停留。
